# My First Webpage Project

A simple two-page website built to practice HTML and CSS fundamentals.

**Author:** Danson Karanja

## Description

This project contains:
- **index.html** — A personal introduction page with lists and links
- **favorite-places.html** — A page about favorite travel destinations with images
- **css/styles.css** — A single shared stylesheet used by both pages

## Technologies Used

- HTML
- CSS

## Setup Instructions

1. Clone this repository
2. Open `index.html` in any web browser
3. Navigate between pages using the links

## CSS Properties Used

The stylesheet demonstrates the following CSS properties as required:
- `color` — Sets text colors
- `text-align` — Centers the main heading
- `font-style` — Controls italic/normal text
- `font-weight` — Sets boldness of text
- `font-family` — Uses Arial/sans-serif for readability
- `font-size` — Sets appropriate sizes for all text elements
- `line-height` — Ensures comfortable reading spacing

## How to Debug

If styles don't appear:

1. Right-click the page and select **Inspect**
2. Go to the **Console** tab (Ctrl+Shift+J on Windows, Cmd+Option+J on Mac)
3. Look for any errors — most common is a wrong path to the CSS file
4. Make sure `css/styles.css` exists and the link in HTML reads `href="css/styles.css"`
